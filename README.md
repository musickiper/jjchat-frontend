## Documentation

### Tach Stack
#### Front
- HTML
- CSS
- JavaScript
- React
- Apollo GraphQL (Boost, Hooks)
- Material UI
- Styled-Component with SCSS
- Firebase (Storage)

#### Back
- Express
- Apollo GraphQL (YOGA)
- GraphQL Subscription (Real-time)
- Passport
- JWT
- Prisma ORM
- My SQL
- Heroku

#### Database
- MySQL
- Prisma ORM

### Functionality (requirements)
* User Login
* Chatting with a friend
* Chatting with multiple friends
* Send an image as a chat
* ..
